# Sample Images

Place your test images in this directory.

## Recommended Images
- Natural scenes with distinct regions
- Medical images (X-rays, MRI scans)
- Satellite/aerial images
- Objects with clear boundaries
- Textured surfaces

You can use images from:
- Personal collection
- Public datasets (Kaggle, ImageNet, etc.)
- Stock photo websites (Unsplash, Pexels)
